# 📚 Examples

A collection of sample integrations and code snippets for the Global Micro-Remittance Bridge.

## 📂 Examples
- **React Example:** How to build a checkout flow.
- **Vue Example:** Integration with Vue.js dashboards.
- **Next.js Example:** Server-side payment handling.
- **Express Example:** Backend-to-backend API integration.
- **Webhook Example:** How to listen for settlement events.

## 🚀 Usage
Navigate to the specific folder and follow the local setup instructions.
