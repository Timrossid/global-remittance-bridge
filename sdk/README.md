# 🛠️ SDK

Official developer tools for integrating the Global Micro-Remittance Bridge into third-party applications.

## 📦 Packages
- **TypeScript SDK:** Full type-safe client for the Payment API.
- **JavaScript SDK:** Lightweight wrapper for web integrations.

## 🚀 Key Functions
- `createPayment()`: Initiate a remittance request.
- `getTransaction()`: Poll for payment status.
- `withdraw()`: Trigger a fund withdrawal.
- `watchPayment()`: Webhook/Polling helper for transaction status.
