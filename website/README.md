# 🌐 Project Website

The marketing and developer portal for the Global Micro-Remittance Bridge.

## 🚀 Pages

- **Home:** High-level value proposition for SMEs and a tour of the platform.
- **Pricing:** Transparent fee structure and value comparison.
- **Developers:** Quick-start guides, API documentation, and SDK links.
- **Roadmap:** Future milestones, vision, and community involvement.
- **About Us:** Our mission to democratize global trade.

## 🛠️ Tech Stack

- **Framework:** Next.js 14 (App Router)
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **Deployment:** Vercel

## 🛠️ Development

To run the website locally:

```bash
npm install
npm run dev
```
